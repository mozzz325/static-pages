肯尼亚轮胎标准选用培训速查卡

适用于轮胎标准初审：先看品名，再看侧标/花纹 Marking，再看规格，最后用 HS 交叉复核。口诀是“人车路、工农场、胎管带圈阀分开判”。

1. 先看品名Tyre、Tube、Flap、Rim/Wheel、Valve 先分开；配件不要套轮胎标准。
2. 再看用途词Passenger、Truck/Bus、OTR、Tractor、Forklift、Backhoe 直接决定大方向。
3. 再看 MarkingLT/C、M/C、R-1/R-4、E/L/G、IND、NHS、REGROOVABLE 都是强线索。
4. 最后看规格规格只能做初判：17.5-25 像 OTR，16.9-28 像农机，11R22.5 像卡客车。
一句话记忆：乘用看 EAS 358，轻卡看 EAS 359，卡客看 EAS 357，工程 OTR 看 4250，农机看 4251/8664，叉车工业充气才看 3739。

公路车辆轮胎

PCR / Car

轿车 / 乘用车胎

Passenger carPCRSUV 无 LT/C

看什么：品名写 Passenger、Car、PCR；规格像 195/65R15 91H、205/55R16。

Marking：P、RADIAL、速度级 H/V/W；SUV 的 HT/AT/MT 只是花纹，不自动变轻卡。

标准桶：KS EAS 358:2004
LT / C

轻卡 / 面包车 / 皮卡胎

Light truckVanCommercial

看什么：规格带 LT 或 C，比如 LT245/75R16、195R15C、205/75R16C。

Marking：LT、C、8PR/10PR、Load Range C/D/E 是强线索。

标准桶：EAS 359:2004，系统内按 Kenya adopted 名称确认
TBR

卡车 / 公共汽车胎

TruckBusTBR

看什么：11R22.5、295/80R22.5、315/80R22.5、12.00R20。

Marking：REGROOVABLE、TUBELESS、位置词 Steer/Drive/Trailer 常见。

标准桶：KS EAS 357:2004；尺寸/轮辋参考 KS ISO 4209-1:2001
M/C

摩托车 / 踏板车胎

MotorcycleScooterM/C

看什么：90/90-17 M/C、2.75-18、3.00-10、120/70ZR17。

Marking：M/C 是 motorcycle；NHS 表示非公路用途。

标准桶：KS ISO 5751-1/2/3:2010；小轮径 code 系列看 KS EAS 475-1:2008 / KS ISO 6054-1
Bike

自行车胎

BicycleBike700C

看什么：700x25C、26x1.95、37-622、28-622。

Marking：ETRTO 双数字如 37-622 是自行车常见写法。

标准桶：KS ISO 5775-1:2023
Retread

翻新胎

RetreadedRemouldedRenewed

看什么：品名或文件明确写 retread/remould/renewed。

Marking：RETREAD、REMOLD、翻新厂标识；新胎标准不能直接替代。

标准桶：KS EAS 53:2000

工程机械 / 工业车辆轮胎

OTR

OTR / Earthmover 土方胎

OTREarthmoverLoaderMining

看什么：17.5-25、20.5-25、23.5-25、14.00-24、16.00R25。

Marking：E-2/E-3/E-4、L-3/L-5、G-2/G-3 是工程用途强线索。

标准桶：KS ISO 4250-1:2023; KS ISO 4250-2:2023
Crane

移动吊车 / 特种高速工程胎

Mobile craneAll terrain craneCrash tender

看什么：385/95R25、445/95R25、525/80R25 等，常有较高速度级。

Marking：Crane、E/F 速度级、公路行驶能力是重点。

标准桶：KS ISO 10571:2016；如接受 ISO 最新，核 ISO 10571:2024
Solid

叉车 / 工业实心胎

ForkliftSolid tyrePress-on

看什么：品名写 Solid、Resilient solid、Press-on solid。

Marking：SOLID、non-pneumatic；注意不是充气胎。

标准桶：KS ISO 10499-1:2019; KS ISO 10499-2:1998；圆柱/锥底看 KS ISO 10500:1991
R-4 / IND

Backhoe / 工程农用型胎

Backhoe loaderIndustrial tractorR-4IND

看什么：12.5/80-18、16.9-24 R-4、19.5L-24。

Marking：R-4 或 IND 是关键；外形像农机，但用途在工程机械。

标准桶：KS ISO 18808:2021
3739

工业车辆 / 叉车充气胎

Industrial tyreForklift pneumaticMaterial handling

看什么：6.50-10、7.00-12、8.15-15、250-15，且品名明确工业车辆。

Marking：Industrial、Forklift、Pneumatic；没有 OTR/E/L/G 工程花纹代码。

标准桶：KS ISO 3739-1:2022; KS ISO 3739-2:2021
Forestry

林业 / 伐木胎

ForestryLoggingSkidder

看什么：品名写 Forestry、Logging、Skidder。

Marking：LS、LS-2、LS-3、Forestry service 是强线索。

标准桶：KS ISO 18807:2019

农业轮胎

R-1W

农业拖拉机斜交胎

Agricultural tractorFarm tractorBias

看什么：16.9-28、14.9-30、18.4-34、13.6-28，常带 PR。

Marking：R-1、R-1W、R-2、F-2；人字花纹是常见视觉线索。

标准桶：KS ISO 4251-1:2019; KS ISO 4251-2:2019
480/70R34

农业径向胎 480/70R34

Tractor radial480/70R34RADIAL

看什么：480/70R34、70 series、Tractor Radial。

Marking：RADIAL、R-1/R-1W、载重指数/速度级比 PR 更常见。

标准桶：KS ISO 8664:2018
520/85R38

农业径向胎 520/85R38

Tractor radial520/85R38R-1

看什么：520/85R38、Agricultural Radial、Rear Tractor。

Marking：RADIAL、R-1、载重/速度符号；规格通常由侧标或技术页确认。

标准桶：KS ISO 8664:2018
IF/VF

农业径向胎 IF/VF

Very High FlexionVF480/70R34RADIAL

看什么：IF 或 VF 前缀，比如 VF480/70R34、IF520/85R38。

Marking：IF/VF 是高挠曲农业径向胎线索，需和普通 radial 区分记录。

标准桶：KS ISO 8664:2018
Implement

农机具 / 农用拖车胎

ImplementFarm trailerI-1

看什么：品名写 Agricultural implement、Farm trailer。

Marking：I-1、11L-15、10.0/75-15.3 等常见；不是拖拉机驱动胎。

标准桶：KS EAS 360:2004
R-1 / R-4

农用胎快速复核

R-1/R-2PRTractor

看什么：如果文件只写 Tyre + 16.9-28，先问用途/花纹；R-1/R-2 走农机，R-4/IND 转工程农用。

Marking：同一尺寸可能对应不同用途，花纹代码比“看起来像”更可靠。

标准桶：普通农机 4251；R-4/IND 18808；径向 8664

配件和轮辋：不要套错“Tyre”标准

Tube

内胎

TubeInner tubeTT tube

看什么：Tube 17.5-25、Tube 23.5-25；HS 常在 4013。

Marking：TR13/TR15/TR218A 等气门嘴型号可辅助判断。

标准桶：KS 700:1990
Flap

垫带 / Rim Flap

FlapRubber flapRim flap

看什么：Flap 17.5-25、Flap 23.5-25；通常跟 TT 工程胎配套。

Marking：只要写 Flap，就不要按 inner tube 审。

标准桶：ISO 21634:2022
Rim / Wheel

钢圈 / 轮辋 / 车轮

RimWheelSteel wheel

看什么：14.0-25 wheel、19.5-25 rim；图片是金属件，不是橡胶胎。

Marking：多件式锁圈、轮辋宽度 x 直径；不要把 Tube 写进品名。

标准桶：OTR 钢圈 ISO 4250-3:2020；农林工程轮辋 KS ISO 18804:2021；卡客车轮辋 KS ISO 4209-2:2020
Valve

气门嘴 / 阀

ValveTyre valveTubeless valve

看什么：TR13、TR218A、V3-06-5、Tubeless valve。

Marking：这是轮胎配件，不按 Tyre/Tube 标准。

标准桶：KS ISO 9413:2019、KS ISO 4570:2002、KS ISO 14960-1/2
